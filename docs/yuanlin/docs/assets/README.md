# docs/assets

## 概述

项目文档所使用的静态图片资源目录，包含截图、界面预览和主题展示图片。这些图片用于文档中的可视化说明，涵盖了 Gemini CLI 的功能截图、监控面板预览以及各种终端主题的展示效果。
