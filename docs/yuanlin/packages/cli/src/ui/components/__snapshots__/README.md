# __snapshots__ (packages/cli/src/ui/components/)

## 概述

此目录为 `packages/cli/src/ui/components/` 下的测试快照目录，是所有快照目录中文件数量最多的一个。用于存储各类 UI 组件的测试快照，涵盖对话框、表单、页头页脚、设置面板、工具确认队列等大量组件。包含 `.snap` 文本快照和 `.snap.svg` 可视化快照两种格式。
