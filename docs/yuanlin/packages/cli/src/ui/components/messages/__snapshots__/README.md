# __snapshots__ (packages/cli/src/ui/components/messages/)

## 概述

此目录为 `packages/cli/src/ui/components/messages/` 下的测试快照目录，用于存储消息类 UI 组件的测试快照。涵盖 Diff 渲染、错误消息、Gemini 消息、工具确认消息、思考消息、用户消息等多种消息组件。包含 `.snap` 文本快照和 `.snap.svg` 可视化快照两种格式。
