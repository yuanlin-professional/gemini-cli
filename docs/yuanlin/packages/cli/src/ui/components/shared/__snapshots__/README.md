# __snapshots__ (packages/cli/src/ui/components/shared/)

## 概述

此目录为 `packages/cli/src/ui/components/shared/` 下的测试快照目录，用于存储共享/通用 UI 组件的测试快照。涵盖选择列表、单选按钮、枚举选择器、可展开文本、滚动容器、搜索列表、虚拟化列表等基础组件。包含 `.snap` 文本快照和 `.snap.svg` 可视化快照两种格式。
