# __snapshots__ (packages/cli/src/ui/)

## 概述

此目录为 `packages/cli/src/ui/` 下的测试快照目录，用于存储 UI 层顶层组件（App、ToolConfirmationFullFrame）的测试快照。包含 `.snap` 文本快照和 `.snap.svg` 可视化快照两种格式。

## 目录结构
