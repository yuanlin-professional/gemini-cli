# __snapshots__ (packages/cli/src/ui/auth/)

## 概述

此目录为 `packages/cli/src/ui/auth/` 下的测试快照目录，用于存储认证相关 UI 组件的测试快照，涵盖 API 认证对话框、登录对话框、账号封禁对话框及 Google 登录重启对话框等。

## 目录结构

```
__snapshots__/
├── ApiAuthDialog.test.tsx.snap                # API 认证对话框快照
├── AuthDialog.test.tsx.snap                   # 认证对话框快照
├── BannedAccountDialog.test.tsx.snap          # 账号封禁对话框快照
└── LoginWithGoogleRestartDialog.test.tsx.snap # Google 登录重启对话框快照
```
