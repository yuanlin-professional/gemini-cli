# commands

## 概述

`commands` 是 `custom-commands` 扩展的命令定义根目录。该目录采用文件系统层级结构来组织自定义命令，每个子目录代表一个命令命名空间，子目录内的 `.toml` 文件定义具体的命令逻辑。Gemini CLI 会自动扫描此目录树，将目录路径映射为命令的层级名称（例如 `fs/grep-code` 对应 `/fs grep-code` 命令）。

## 目录结构

```
commands/
└── fs/                    # 文件系统相关命令的命名空间子目录
    └── grep-code.toml     # 代码搜索命令定义
```

## 核心组件

- **fs/** - 文件系统操作命令组。包含与文件搜索、内容检索相关的自定义命令。命令通过 TOML 格式定义，支持模板变量 `{{args}}` 接收用户输入参数，并可通过 `!{...}` 语法嵌入 shell 命令执行结果。
