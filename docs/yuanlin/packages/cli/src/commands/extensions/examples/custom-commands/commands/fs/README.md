# fs

## 概述

`fs` 是自定义命令中的文件系统命名空间目录，包含与文件系统操作相关的命令定义。该目录下的每个 `.toml` 文件定义一个可被 Gemini CLI 调用的自定义命令，命令通过 prompt 模板将用户参数与 shell 命令执行结果组合，发送给模型进行处理。

## 目录结构

```
fs/
└── grep-code.toml    # 代码 grep 搜索命令
```

## 核心组件

- **grep-code.toml** - 代码搜索命令定义文件。该命令接收用户传入的搜索模式 `{{args}}`，通过内嵌的 `grep -r {{args}} .` shell 命令在当前目录递归搜索匹配内容，然后将搜索结果作为上下文注入 prompt，请求模型对搜索结果进行总结。这展示了自定义命令的核心模式：模板参数 + shell 命令嵌入 + 模型 prompt。
